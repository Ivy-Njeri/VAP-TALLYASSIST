💼 What is Payroll?

Payroll refers to the process of managing employee compensation, including salaries, wages, bonuses, deductions, and net pay. Tally simplifies payroll by automating salary calculations, statutory deductions, payslips, and reports.

🎯 Why is Payroll Important?

  • Ensures accurate and timely salary processing
  • Helps in compliance with statutory requirements like PAYE, NHIF, NSSF
  • Keeps employee records organized
  • Automates repetitive calculations

⚙️ How to Enable Payroll in Tally

  1. Go to Gateway of Tally → F11: Features
  2. Enable Payroll Features → Set to Yes
  3. Press Ctrl + A to save

🧾 Payroll Masters in Tally

Employee Creation:

  1. Go to Payroll Info → Employees → Create
  2. Enter employee name, designation, group, and join date

Payroll Units (Optional):

  • Used for attendance or work-based payroll (e.g., hours, days, pieces)
  • Create from Payroll Info → Units

Salary Details Setup:

  1. Go to Payroll Info → Salary Details → Create
  2. Select employee and define earnings/deductions like Basic Pay, Allowances, PAYE

📅 Attendance / Production Types

  • Create from Payroll Info → Attendance / Production Types
  • Examples: Present, Absent, Leave, Overtime

🧾 Processing Payroll

  1. Go to Payroll Vouchers
  2. Choose the month and employee(s)
  3. System will auto-calculate based on setup
  4. Save voucher with Ctrl + A

📄 Payslip & Reports

  • Payslip: Go to Display → Payroll Reports → Payslip
  • Payroll Statement: Summary of payroll by month
  • Employee Ledger: Detailed payroll history
  • Pay Sheet: View and export salary sheet

📌 Tips for Payroll in Tally

  • Ensure correct grouping of employees (e.g., Staff, Management)
  • Backup before monthly processing
  • Use separate ledgers for each salary component
  • Reconcile with statutory returns (KRA, NSSF, NHIF)
